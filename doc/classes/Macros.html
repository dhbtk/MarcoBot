add_macro(channel,trigger,macro)
check_for_macro(message,target,user)

Checks for a macro on the message. If found, and the timeout >= timeout set in the settings file, it is outputted.

macrocheck(message,target)

Does the same as check_for_macro, but displays it only when asked for and says the timeout time too. Rather neat.

macrologic(message)

Macrologic checks for macros on that message. It returns an array with the trigger, the macro and the time since the last activation. Is used by macrocheck() and check_for_macro().
